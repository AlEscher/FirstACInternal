# FirstACInternal
My first internal cheat for the game [Assault Cube](https://assault.cubers.net/)

## Features
* Godmode
* Unlimited Ammo
* No recoil
* Rapid fire
* Unlimted grenades
* SpeedHack (currently unstable)
* Set teleporter coordinates
* Teleport to set coordinates
* Add 10 kills to scoreboard
